## Multi-Language Support

Sell globally without language barriers. Create multi-language versions of every checkout block so each customer sees content in their own language.

### How It Works

For every block you create, add translations for any language. Checkout Ninja detects the customer's language from their Shopify Markets settings and displays the right version automatically.

### Key Features

- **Unlimited languages** — Add as many language versions as you need for each block
- **Shopify Markets integration** — Automatically detects the customer's market and language
- **Per-block translations** — Translate individual blocks independently — no need to duplicate entire configurations
- **Fallback language** — Set a default language for customers whose language isn't configured
- **RTL support** — Full right-to-left layout support for Arabic, Hebrew, and other RTL languages
